\doxysubsection{Detailed Description}
Sprite management for the game This module provides functions to create, destroy, and draw sprites. 

It also defines the sprite type, which can be FULL\+\_\+\+IMAGE, BY\+\_\+\+LINE, or BY\+\_\+\+PIXEL.
