\doxysubsection{Detailed Description}
Sprite loader for the game. 

This module provides functions to load and manage sprites used in the game. It includes functions to load player sprites, bomb sprites, wall sprites, background sprites, and menu sprites. It also provides functions to retrieve specific sprites for different game elements.
